# Changelog

All notable changes to Markora will be documented in this file.

## 0.1.0 - 2026-06-16

### Added

- Initial Electron, React, TypeScript, and Vite application scaffold.
- Typora-inspired shell with titlebar, sidebar, editor page, outline, status bar, menus, themes, and focus mode.
- Tiptap/ProseMirror WYSIWYG Markdown editor.
- Markdown conversion pipeline with Source Code Mode.
- YAML front matter, task lists, KaTeX math, Mermaid diagrams, and PlantUML placeholder rendering.
- File open, folder open, save, save-as, workspace scanning, image paste/drop, and local asset import.
- Table insertion and contextual table editing controls.
- HTML, PDF, and Word-compatible `.doc` export.
- Automated lint, unit, renderer build, and browser smoke verification.
